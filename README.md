 Alerta Tubarão

Aplicação web desenvolvida com React, Tailwind CSS e AWS, com foco em estudo prático de desenvolvimento front-end, integração com serviços em nuvem e arquitetura moderna.

O projeto simula um sistema de alerta e gerenciamento de ocorrências, aplicando conceitos reais usados no mercado.

---

 Tecnologias Utilizadas

- React – construção da interface
- Tailwind CSS – estilização e layout responsivo
- JavaScript
- AWS – uso de serviços em nuvem (Lambda, API Gateway, etc.)
- Git & GitHub – versionamento de código
- Vercel – deploy do front-end

---

 Funcionalidades

- Interface web responsiva
- Organização de componentes React
- Integração com backend em AWS
- Comunicação com funções **AWS Lambda**
- Estrutura pensada para uso real (front-end + cloud)

⚠️ **Observação:**  
Algumas funcionalidades exigem **login administrativo** ou estão limitadas devido ao ambiente **AWS Sandbox**, sendo usadas apenas para fins de demonstração e estudo.
